# Gen-Ai-MSME_form


## Set Up the Backend:
#### Install dependencies in the backend directory:
```bash
cd backend
pip install -r requirements.txt
```
####  Start the Flask server:
```bash
python app.py
```

## Set Up the Frontend:

#### Install dependencies in the frontend directory:
```bash
cd frontend
npm install
```

#### Start the React application:
```bash
npm start
```
